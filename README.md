# 📊 Big Mart Sales Prediction

Big Mart Sales Prediction is a **Machine Learning project** that predicts product sales for retail stores using historical sales data and various product and outlet attributes. The project applies data preprocessing, exploratory data analysis, feature engineering, and regression algorithms to build an accurate sales prediction model.

---

## 📌 Project Overview

The objective of this project is to help retail businesses estimate future product sales by analyzing historical data and identifying key factors that influence sales performance.

The application demonstrates a complete Machine Learning workflow, including:

* 📂 Data Collection
* 🧹 Data Cleaning & Preprocessing
* 📊 Exploratory Data Analysis (EDA)
* ⚙️ Feature Engineering
* 🤖 Model Training
* 📈 Sales Prediction
* 📉 Model Evaluation
